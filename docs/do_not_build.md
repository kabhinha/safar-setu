# DO NOT BUILD LIST
**Strictly Prohibited Features**

The following features are **BANNED**. Any code found implementing these will cause immediate build failure.

1.  **Social Media Scrapers:** No Twitter/X, Facebook, or Instagram scraping.
2.  **GPS/Map Integration:** No Google Maps API, no OpenStreetMap with precise markers.
3.  **Face Recognition:** No `face_recognition`, `dlib` facial landmarks, or cloud vision face detection.
4.  **Route Optimization:** No pathfinding or navigation features.
5.  **Biometric Storage:** No fingerprints, iris scans, or voiceprints.
6.  **Public Sign-up:** No "Register" button for the general public.
7.  **Raw Footage Recording:** No `.mp4` or `.avi` storage of CCTV feeds.
